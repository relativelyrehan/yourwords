# note-taking

## General info
This project is simple thought/message sharing canvas.
	
## Technologies
Project is created with:
* Node
* Express
* EJS
* MongoDB

	
## Setup
To run this project, install it locally using npm:
* npm install
### run
* npm run start

### run development
* npm run dev
